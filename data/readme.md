## The LTNs Encoding of the Visual Relationship Dataset

[Here](
https://drive.google.com/drive/folders/14TxGKJPPvU0f15XFi_F9_86AJ6WvwXqr?usp=sharing) you can download the `FFoCat.zip` file, unzip it in your local machine. The dataset is already divided into the `train` and `test` folder. The file `label.tsv` contains the food labels, the file `food_food_category_map.tsv` contains the food labels with the corresponding food category labels.
